# Youchuan AI API

Welcome to the Youchuan AI API documentation. Our comprehensive suite of AI-powered image and video generation tools enables you to create, modify, and enhance visual content with cutting-edge artificial intelligence.

## Getting Started

The Youchuan AI API provides powerful image generation and manipulation capabilities through simple REST endpoints. All requests require proper authentication using your API credentials.

### Authentication

All API requests require the following headers:

- `x-youchuan-app`: Your application identifier
- `x-youchuan-secret`: Your secret key

## API Base URL

```
https://ali.youchuan.cn
```

## Quick Start Example

Generate your first image with a simple API call:

```bash
curl -X POST "https://ali.youchuan.cn/v1/tob/diffusion" \
  -H "x-youchuan-app: YOUR_APP_ID" \
  -H "x-youchuan-secret: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "A beautiful sunset over mountains with vibrant colors"
  }'
```

## Response Format

All endpoints return a Job object containing:

- `id`: Unique job identifier
- `status`: Processing status (0: created, 1: processing, 2: completed, 3: failed)
- `urls`: Array of result image URLs (populated when completed)
- `cost`: Credit consumption details
