# Video Extend

Extend the duration of existing videos by generating additional frames that continue the motion and narrative.

## Endpoint

```
POST /v1/tob/extend-video
```

## Headers

| Header | Type | Required | Description |
|--------|------|----------|-------------|
| `x-youchuan-secret` | string | Yes | Your Youchuan secret key |
| `x-youchuan-app` | string | Yes | Your Youchuan app identifier |

## Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `jobId` | string | Yes | ID of the original video generation task |
| `videoNo` | integer | Yes | Video number to extend (0/1/2/3) |
| `prompt` | string | No | Text prompt to guide the extension (1-8192 characters) |
| `callback` | string | No | Callback URL for task completion notifications |

## Example Request

```json
{
  "jobId": "job_video_12345",
  "videoNo": 0,
  "prompt": "Continue the ocean waves gently rolling onto the shore",
  "callback": "https://your-domain.com/webhook"
}
```

## Response

Returns a [Job object](/api/schemas/job) containing the extended video task information.

```json
{
  "id": "job_video_extend_67890",
  "status": 0,
  "text": "Continue the ocean waves gently rolling onto the shore",
  "urls": [],
  "cost": {
    "feeCost": 200,
    "relaxCost": 0
  }
}
```

## Status Codes

| Code | Description |
|------|-------------|
| 200 | Success |
| 400 | Invalid prompt parameter |
| 401 | Invalid app identifier |
| 402 | Insufficient account balance |
| 403 | Prompt contains sensitive content |
| 405 | Video can only be extended 4 times maximum |
| 429 | Maximum concurrent limit reached |
| 500 | Internal server error |

## Extension Limits

- **Maximum Extensions**: Each video can be extended up to **4 times**
- **Seamless Continuity**: Extensions maintain visual and motion continuity with the source video
- **Progressive Quality**: Multiple extensions may gradually reduce quality

## Use Cases

- **Storytelling**: Extend narrative sequences for longer stories
- **Looping Content**: Create longer loops from short clips
- **Scene Development**: Allow scenes to develop and evolve over time
- **Content Adaptation**: Adjust video length for specific platform requirements

## Notes

- **Quality Preservation**: Each extension attempts to maintain the visual style and motion patterns
- **Context Awareness**: The AI analyzes the ending frames to create coherent continuations
- **Prompt Guidance**: Use descriptive prompts to guide how the video should continue
- **Processing Time**: Extensions require additional processing time based on the original video complexity
- **File Management**: Extended videos are provided as new files, not appended to originals
